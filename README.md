# Vintage Point-and-Shoot Camera

A retro-style digital point-and-shoot camera built using an ESP32-WROVER and MicroPython.
This project focuses on embedded systems, hardware/software integration, and product-oriented design while creating a minimalist digital camera experience.

---
## Project Goals

- Build a functional digital camera using embedded hardware
- Create a retro-inspired point-and-shoot experience
- Learn MicroPython and ESP32 development
- Practice embedded systems engineering
- Document the full engineering process
---
## Current Features

- ESP32 camera initialization
- Browser-based live camera stream
- WiFi connectivity
- Modular MicroPython project structure
---
## Planned Features

- TFT live preview screen
- Physical shutter button
- SD card photo storage
- Photo counter
- Battery indicator
- Portable battery-powered operation
- Retro camera enclosure
---
## Hardware Used

| Component | Purpose |
|---|---|
| ESP32-WROVER | Main controller |
| OV2640 Camera | Image capture |
| TFT Display | Live preview |
| Push Button | Shutter control |
| MicroSD Card | Photo storage |
| Breadboard + Jumper Wires | Prototyping |
